# Dressage app
Simple quiz learner.

## Requirements
* python==3.5
* Flask==0.12
* others see in [requirements](requirements.txt)

## Run app
* install python, virtualenv
* virtualenv dressage_env
* pip install requirements.txt
* python db_create.py
* python db_migrate.py
* python filler.py
* python run.py
